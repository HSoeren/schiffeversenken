# Schiffe Versenken

.. als Bogenscheibe (80x80 cm)

Bisher immer in einem Grafikprogramm erzeugt, was recht aufwändig ist. Als simple Webseite mit JavaScript kann jetzt jederzeit eine neue Scheibe erstellt werden.
